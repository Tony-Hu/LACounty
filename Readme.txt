Steps to run the application.
	1. The following PDF files should be put in the same directory of the application before running the application.
		CR180
		CR181
		FW001
		FW003
		POS040
		CR115

	2. Run InfoReader.java and enter user input.

	3. The PDF forms will be filled in the same directory of the application.
